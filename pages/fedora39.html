Fedora 39 Released

Distro Updates

Fedora 39, the latest release of the popular Linux distribution, was released on November 7, 2023. The release comes with a variety of new features and improvements, making it one of the most exciting Fedora releases in recent years.

One of the most exciting new features of Fedora 39 is a new immutable variant called Fedora Onyx. This variant uses an immutable file system, meaning that files on the system cannot be modified without special permission. This can help prevent malware attacks and user errors.

Fedora 39 also includes a variety of new Flatpak applications, including Blender 3.2.3, Inkscape 1.2.1, and LibreOffice 7.3.2. These applications offer a variety of new features and improvements, making them attractive options for users looking for modern and powerful applications.

In addition, Fedora 39 has seen significant performance improvements, especially on systems with multi-core processors. This is thanks to the use of the new Linux kernel 5.18 and performance improvements to the built-in applications.

Overall, Fedora 39 is a solid release that offers a variety of new features and improvements. The release is suitable for users looking for a fast, stable, and secure Linux operating system.

Highlights of the Fedora 39 release:

  • Fedora Onyx: An immutable variant for a more secure and stable experience.
  • New Flatpak applications:
    • Blender 3.2.3: A popular 3D modeling and animation application.
    • Inkscape 1.2.1: A popular vector graphics application.
    • LibreOffice 7.3.2: A popular open-source office suite.
  • Significant performance improvements: Especially on systems with multi-core processors.

Details of the new features and improvements:

Fedora Onyx

Fedora Onyx is a new immutable variant of Fedora that uses an immutable file system. This means that files on the system cannot be modified without special permission. This can help prevent malware attacks and user errors.

To use Fedora Onyx, you will need to create a new user account and select the "Onyx" option when installing Fedora.

New Flatpak applications

Fedora 39 includes a variety of new Flatpak applications, including:

  • Blender 3.2.3: A popular 3D modeling and animation application.
  • Inkscape 1.2.1: A popular vector graphics application.
  • LibreOffice 7.6.3: A popular open-source office suite.

These applications offer a variety of new features and improvements, making them attractive options for users looking for modern and powerful applications.

Performance improvements

Fedora 39 has seen significant performance improvements, especially on systems with multi-core processors. This is thanks to the use of the new Linux kernel 6.5 and performance improvements to the built-in applications.

For example, the GNOME Shell desktop environment now starts up 15% faster than in Fedora 38.

Overall assessment

Fedora 39 is a promising release that offers a variety of new features and improvements. The release is suitable for users looking for a fast, stable, and secure Linux operating system.

We are particularly impressed with the new Fedora Onyx variant, which offers a more secure and stable experience. We are also pleased with the new Flatpak applications, which offer a variety of new features and improvements.

We recommend Fedora 39 to users who are looking for a modern and powerful Linux operating system.
